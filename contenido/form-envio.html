¿Quieres enviar tu servidor para que se suba a la página?

Aqui te enseño a como subirlo.



  • Nombre del servidor: En este apartado se debe escribir el nombre/titulo del servidor que quieras publicar. (max.50)

  • Descripción: En este apartado se debe poner cualquier información del servidor configurado, ya sea la lista de modalidades, plugins/addons o cualquier otra cosa. (max.500)

  • Link de descarga: Aqui va el link de descarga del servidor a publicar, para que un tester descargue y compruebe que tu servidor sea funcional. (max.200)

  • Correo Electronico: Debes escribir un correo electronico que uses, ya que se te enviara un correo con un "ID de Usuario" y "ID de Server" para cuando quieras actualizar los datos de tu servidor que publicaste. (max.100)

  • -- Selecciona el Software --: Aqui se debe seleccionar el tipo de software y versión que ocupa el servidor que estas enviando.

  • Aceptó los Terminos: Este apartado se debe leer, comprender y aceptar.

    • Y para finalizar, debes presionar el botón "Enviar Servidor".
