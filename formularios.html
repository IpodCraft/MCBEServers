¿Quieres actualizar un servidor que publicaste con anterioridad?

Aqui te enseño a como actualizar los datos.



  • Nombre del servidor: En este apartador puedes ocupar el mismo nombre del servidor o cambiarlo. (max.50)

  • Descripción: Aqui puedes actualizar la descripción de tu servidor. (max.500)

  • Link de descarga: Aqui va el link de descarga del servidor a actualizar, para que un tester descargue y compruebe que tu servidor sea funcional. (max.200)

  • Correo Electronico: Si ya no cuentas con el correo electronico con el que enviaste el servidor que quieres actualizar, puedes ocupar uno diferente, ya que para comprobar que es tu servidor, se debe usar el "ID de Usuario" y "ID de Server" que se te envio cuando fue publicado. (max.100)

  • ID de Usuario: Aqui se debe poner el ID de Usuario que se te proporcionó en tu correo electronico cuando enviaste por primera vez el servidor que estas por actualizar. (Este dato es privado)

  • ID de Server: Aqui se debe poner el ID de Server que se te proporcionó en tu correo electronico cuando enviaste por primera vez el servidor que estas por actualizar. (Este dato es privado)

  • -- Selecciona el Software --: Aqui se debe seleccionar el tipo de software y versión que ocupa el servidor que estas actualizando.

  • Aceptó los Terminos: Este apartado se debe leer, comprender y aceptar.

    • Y para finalizar, debes presionar el botón "Actualizar Servidor".
